# Scientific Calculator App

A modern, dark-mode scientific calculator built using HTML, CSS, and JavaScript.  
This calculator supports both basic and advanced scientific operations with a clean and responsive UI.

Live Demo:  
https://anjanajanardhanan-hub.github.io/calculator-app/

---

## Features
- Dark-mode user interface  
- Basic operations: +, -, ×, ÷  
- Scientific functions:  
  - sin, cos, tan  
  - log10  
  - Square root  
  - pi (π) and e constants  
  - x² and xⁿ power functions  
- Brackets support ( )  
- Smooth button animations  
- Fully responsive (mobile and desktop)

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla)

---

## How to Run Locally
1. Download the repository  
2. Open the file: index.html  
3. The calculator will open in your browser
